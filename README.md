📦 E-Commerce Shipping Dashboard

An interactive Power BI dashboard designed to analyze shipping performance, delivery efficiency, and customer behavior in an e-commerce environment.

🚀 Project Overview

This project focuses on extracting meaningful insights from shipping and customer data to improve delivery operations and customer satisfaction.

The dashboard provides a comprehensive view of:

Order distribution
Delivery performance (On-Time vs Late)
Customer behavior & ratings
Warehouse efficiency

📈 Insights Derived
Nearly 50% of deliveries are late, indicating operational inefficiencies
Certain warehouses (e.g., F, C) show higher late delivery rates
Customer rating is moderately low (avg 3.0), possibly linked to delays
Higher customer care calls indicate delivery-related issues
Delivery performance varies significantly across shipment modes
